CAILY CAFE 凱莉日嚐

店長可以在這裡新增、修改、刪除商品。若已設定 Google Apps Script，送出後會同步到 Google Sheet。

新增 / 修改商品

如果輸入新的商品 ID，就是新增商品；如果輸入已存在的商品 ID，就是修改該商品。

刪除商品

輸入商品 ID 後按下刪除，會送出刪除要求到 Google Sheet。

操作方式

  1. 新增商品：填入新的商品 ID 與資料後送出。
  2. 修改商品：填入同一個商品 ID，再更新資料後送出。
  3. 刪除商品：在右側刪除區輸入商品 ID 後刪除。
  4. 圖片可以先放在網站資料夾，也可以使用網路圖片網址。
